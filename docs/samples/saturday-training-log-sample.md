## 4주 후 자동 매칭 (비포애프터)

같은 견종·보호자가 4주 후 다시 방문 → 봇 메모에 같은 정보 들어오면:

```sql

INSERT INTO training_sessions (..., followup_session_id)
VALUES (..., '<위 INSERT의 id>');
```

→ 자동으로 #18·#19 비포애프터 슬롯에 매칭, "변화 다큐" 캐러셀 1편 자동 생성.
