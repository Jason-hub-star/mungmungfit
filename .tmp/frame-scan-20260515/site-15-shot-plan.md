## Best next saving rule

When saving more images from MP4, prefer:
- a different breed
- a different tool setup
- a different camera angle
- a visible trainer interaction
- a cleaner wide composition

Avoid saving more if it is only:
- another white fluffy dog on the same yellow cavaletti
- another nearly identical front-facing stance
- a blurry frame with only color variation
