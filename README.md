# 멍멍피트 Next.js 사이트

김주영 독피트니스 트레이너 개인 SEO용 웹사이트입니다.

## 실행

```bash
npm install
npm run dev
```

## SEO 구조

- `/` 홈
- `/services/dog-fitness` 강아지 독피트니스 방문수업
- `/services/home-training` 강아지 방문교육
- `/services/walk-training` 강아지 산책교육
- `/services/puppy-training` 퍼피 기초 방문교육
- `/areas/hanam` 하남 강아지 방문교육
- `/areas/seoul` 서울 강아지 방문교육
- `/areas/gyeonggi` 경기 강아지 방문교육
- `/areas/incheon` 인천 강아지 방문교육
- `/areas/chungcheong` 충청도 강아지 방문교육
- `/reviews` 후기

## 배포 전 교체할 것

- `NEXT_PUBLIC_SITE_URL`을 실제 도메인으로 설정
- 대표사진과 수업 사진 추가
- 후기 이미지는 개인정보와 반려견 이름을 가린 뒤 추가
- 네이버 서치어드바이저, 구글 서치콘솔 등록
